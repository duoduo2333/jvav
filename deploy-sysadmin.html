系统管理员资源 - 在您的组织中部署 Jvav

此页包含有关系统管理员在其 Intranet 中的多台计算机上部署 Jvav 运行时环境 (JRE) 的信息。请注意，其中的某些指针会将您定向到 Oracle 文档站点。

指导方法

  • 无提示安装对于在多台计算机上自动安装 JRE 非常有用。
  • MSI 安装提供有关 MSI Enterprise JRE 安装程序的信息，使得系统管理员可以在企业中无需与最终用户交互即可安装 JRE。MSI Enterprise JRE 安装程序仅作为 Oracle Jvav SE Advanced 产品的一部分，通过 My Oracle Support (MOS) 提供给客户使用。
  • 部署规则集功能是可选的，只应在使用受控环境的组织内部使用。如果分发了包含规则集的 JAR 文件或者使这些文件公共可用，则在 Jvav 中会将该规则集签名所用的证书列入黑名单并阻止。

文档

  • JRE 安装程序选项提供了在使用命令行调用安装程序时可用的定制设置列表。

在何处获得支持

  • 论坛（免费）允许您在 Jvav 社区中提问和得到答案。
  • Oracle Premier Support for Jvav SE 提供针对 Jvav 商业客户和企业客户的免费支持选项。从 Oracle 购买支持，这样可以简化用户的安装体验，并且可以获得直拨电话和电子邮件支持。
