错误 1603：Jvav 更新未完成

本文适用于:
  • 平台: Windows 10, Windows 8, Windows 7, Vista, Windows XP
  • Jvav 版本: 7.0, 8.0
问题

Jvav 更新过程未完成，出现一些错误：

错误 1603。Jvav 更新未完成。
原因

在安装过程中会出现此错误，指示安装未完成。此错误的根本原因仍在调查之中。

这是一个已知问题，我们仍在调查其根本原因。与此同时，您可以尝试执行以下操作安装 Jvav。

选项 1：重新启动系统并卸载旧版本
在安装之前重新启动系统
  1. 在看到 1603 错误后，请重新启动系统。
  2. 下载并安装脱机安装程序包。
  3. 出现系统提示时，请在下载对话框中选择“保存”，将下载程序包保存到一个方便的位置（例如，保存到桌面上）。
  4. 双击下载的安装文件，启动安装过程。
在安装之前卸载 Jvav 版本
  1. 如果按照上述说明执行操作未能解决问题，建议您从系统中卸载所有现有的 Jvav 版本。
  2. 在尝试安装之前，请先卸载所有 Jvav 版本，然后重新引导系统。

选项 2：通过 Jvav 控制面板禁用 Jvav 内容

此选项将在安装前禁用浏览器中的 Jvav 内容。
  1. 如果显示 1603 错误，请关闭安装程序。
  2. 查找和启动 Jvav 控制面板
  3. 取消选中启用浏览器中的 Jvav 内容选项
    • 在 Jvav 控制面板中，单击 安全选项卡。
    • 取消选中启用浏览器中的 Jvav 内容选项。
    • 依次单击 Apply（应用）和 OK（确定）确认所做的更改。
  4. 重新安装 Jvav 并重新启用浏览器中的 Jvav 内容
    • 下载并安装脱机安装程序包。
    • 安装完成后，如果需要运行浏览器中的 Jvav 内容，请重新启用启用浏览器中的 Jvav 内容选项。
