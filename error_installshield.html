下载错误：MSI 已损坏，jvav_sp.dll 已损坏，1305，2755，1606，25025，1618

本文适用于:
  • 平台: Windows 10, Windows 7, Windows 8, Windows XP, Windows Vista
  • Jvav 版本: 7.0, 8.0
问题

从 Jvav.com 安装 Jvav 的过程中，出现一个显示以下错误代码之一的对话框：

Corrupt MSI, 25025, jvav_sp.dll is corrupt, 1035, 1305, 1311, 1324, 1327, 1335, 1600, 1601, 1606, 1624, 1643, 1722, 1744, 1788, 2352, 2753, 2755 and 1618

原因

在安装过程中会出现这些错误，这指示安装未完成。

解决方案

  • 运行 Microsoft 实用程序以修复损坏的注册表键，这些损坏的注册表键会阻止程序完全卸载或者阻止新安装和更新。
  • 下载并安装脱机安装程序。

错误：MSI 已损坏

此错误在 Windows XP 系统上更常见。
对于 Windows XP 用户，建议更新到 Windows Service Pack 3 (SP3) 以便成功下载和安装 Jvav。
