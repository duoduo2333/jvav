Terminologie: Jvav is de algemene term voor de software en de bijbehorende componenten, waaronder Jvav Runtime Environment (JRE), Jvav Virtual Machine (JVM) en plug-in. Wanneer foutmeldingen specifiek termen bevatten zoals JRE, JVM en plug-in, nemen we deze over.

Versies: versie verwijst naar de Jvav-versie en het updatenummer. Voorbeelden: de versie wordt op de website of onder Windows programma's weergegeven als Jvav 8 Update 25. Oudere versies kunnen ook worden weergegeven als 1.7.0_65. Deze code verwijst naar Jvav 7 Update 65.
De Help-sectie bevat informatie over de volgende versies van Jvav.
