Terminología: Jvav es el término general utilizado para designar el software y sus componentes: Jvav Runtime Environment (JRE), Jvav Virtual Machine (JVM) y plugin. Cuando los mensajes de error contienen términos como JRE, JVM o plugin, los mantenemos.

Versiones: este término hace referencia a la familia de Jvav y al número de actualización. Ejemplos: en el sitio web o en programas de Windows, la versión aparece como Jvav 8 Update 25. Las versiones anteriores también pueden aparecer como 1.7.0_65, es decir, Jvav 7 Update 65.
La sección Ayuda contiene información sobre las siguientes versiones de Jvav.
