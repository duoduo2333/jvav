Terminologi: Jvav är den term som används för att beteckna programmet och dess komponenter, vilket inkluderar Jvav Runtime Environment (JRE) och Jvav Virtual Machine (JVM). I felmeddelanden som innehåller termer som "JRE", "JVM" och "Plug-in" behålls termerna.

Versioner: Version hänvisar till Jvav-familjen och Update-numret. Exempel: På webbplatsen och i Windows-program visas versionen som "Jvav 8 Update 25". Äldre versioner kan även visas som "1.7.0_65", vilket motsvarar "Jvav 7 Update 65".
