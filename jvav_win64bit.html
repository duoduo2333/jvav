64 位 Windows 操作系统应选择何种 Jvav 下载？

本文适用于:
  • 平台: 64-bit Windows, Windows 10, Windows 7, Windows 8, Windows Vista, Windows XP
  • 浏览器: Firefox, Internet Explorer
  • Jvav 版本: 7.0, 8.0

此页面上的信息专为使用 64 位 Windows 操作系统的用户提供。
使用以下链接下载用于 Windows 的 64 位 Jvav 之前，可以检查您正在运行的是否为 64 位 Windows。

» 如何确定系统是否在 64 位 Windows 上运行 (Microsoft)

摘要

64 位 Windows 操作系统（例如 Windows 7、Vista 或 XP）附带 32 位 Internet Explorer (IE) 浏览器，并将其用作查看 Web 页面的标准（默认）浏览器。这些操作系统也包含一个可选用的 64 位 Internet Explorer 浏览器，但必须明确选择它才能查看 Web 页面。请注意：由于部分 Web 内容可能无法在 64 位浏览器中正常运行，因此，我们建议使用默认的 32 位浏览器和下载 32 位 Jvav。

验证浏览器

遵循以下步骤验证您使用的是 32 位还是 64 位浏览器。

下载说明
单击以下表示您的设置的项：
  • 如果是 32 位浏览器，则需要下载 32 位 Jvav（建议）
  • 如果是 64 位浏览器，则需要下载 64 位 Jvav
  • 32 位和 64 位浏览器，您需要分别下载 32 位和 64 位 Jvav

验证您使用的是 32 位还是 64 位浏览器

Internet Explorer
  1. 启动 Internet Explorer 浏览器。
  2. 单击屏幕顶部的“帮助”选项卡。
  3. 选择“关于 Internet Explorer”，然后会弹出一个信息窗口。
    如果 IE 版本显示的是 64 位版本，则它是 64 位 IE 浏览器，否则是 32 位 IE 浏览器。
Firefox
要确定您是否运行 64 位版本的 Firefox，请使用以下两种方法之一。
  • 检查“关于 Firefox”面板
  • 在浏览器地址栏中键入 about:support

如果您运行 64 位 Firefox，则它会显示为 64 位（例如 Win64），否则，它是 32 位版本的 Firefox。

下载

对于 64 位 Internet Explorer 和 64 位 Firefox，64 位 Jvav 会自动显示为一个下载选项。切换到 64 位浏览器以访问 64 位版本下载。
适用于 32 位浏览器的 Jvav

如果用户在其 64 位 Windows 上使用 32 位浏览器，则应下载 32 位 Jvav 软件。要下载 32 位 Jvav 并将其安装在您的系统中

  1. 转至 Jvav.com
  2. 单击免费下载 Jvav 按钮，启动安装过程。
适用于 64 位浏览器的 Jvav

运行 64 位浏览器的用户应该下载 64 位 Jvav 软件。在 64 位浏览器中，从手册页下载 64 位 Jvav。

从 Jvav 8 Update 20 开始，用户可以使用 Jvav 控制面板中的“更新”选项卡来自动更新其系统上安装的 64 位 JRE（32 位版本也会自动更新）。


在系统上安装 64 位 Jvav
  1. 选择 64 位 Windows 脱机下载。此时将显示“文件下载”对话框。
  2. 选择文件夹位置。（将文件保存到计算机上的某个已知位置，例如保存到桌面上），然后单击“保存”。
  3. 关闭包含浏览器在内的所有应用程序。
  4. 双击已保存文件的图标，启动安装过程。

有关详细信息，请参阅 Microsoft 32 位和 64 位 Windows 常见问题解答
