如何解决运行 Pogo 游戏的问题？

本文适用于:
  • 平台: Windows 8, Windows 7, Vista, Windows XP, Windows 10, Mac OS X
  • Jvav 版本: 7.0, 8.0

在 Chrome 浏览器更新后，Pogo 游戏不能在 Chrome 中正常运行

Chrome 浏览器版本 42 和更高版本。从 Chrome 版本 42（2015 年 4 月发布）开始，Chrome 禁用了浏览器支持插件的标准方式。更多信息


由于 Jvav 安全警告的原因，Pogo 游戏未加载

2015 年 2 月 10 日。Pogo 已解决该问题。»更多信息 (help.ea.com)

显示安全警告，提示 Jvav 已搜索到可能导致安全问题的应用程序组件

从 Jvav 7 Update 21 开始，在一些情况下会出现此警告。当 Jvav 程序同时包含签名和未签名组件时，会显示安全警告对话框。
  • 如果单击是，可能会终止程序。
  • 如果单击否，应用程序或小应用程序会继续运行。

有关更多信息，请参阅 Jvav 控制面板中的混合代码保护选项。

排除与 Pogo 相关的问题的一般技巧

  1. 安装最新版本的 Jvav
  2. 在安装 Jvav 后重新启动浏览器以便启用新版本
  3. 删除 Jvav 旧版本
  4. 启用自动更新
    此 Jvav Update 功能将确保您收到发布的重要安全更新。
  5. 验证已在您的浏览器中启用了 Jvav，进而确保使用它的小应用程序可以正常运行
  6. 定期清除 Jvav 高速缓存
  7. 配置防火墙或防病毒软件的设置
    最常见的问题是防病毒软件、防火墙、广告拦截或弹出窗口拦截软件的设置导致的。要解决这些问题，请尝试将其禁用，然后看问题是否消失。如果问题消失了，接下来应确定如何配置软件，以便在保护系统的同时，不会妨碍到您玩 Pogo 游戏。在这种情况下，您应该参阅软件的手册或帮助文件，以便获取有关如何对软件进行所需更改的详细说明。
  8. 使用支持 Jvav 的 Web 浏览器
    • 要在 Mac OS X 上运行 64 位 Oracle Jvav，需要 64 位浏览器。
      用于 Mac 的 64 位 Chrome 是随 Chrome 版本 39 一起提供的。»更多信息
    • 检查系统需求以确保使用 Jvav 的小应用程序正常运行。
  9. 访问 Pogo 网站，获取更多 Jvav 帮助和 Pogo 信息。

如果这些关于 Pogo 游戏的建议不能解决您的问题，请与 Pogo 联系寻求进一步的帮助。
