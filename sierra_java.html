更新到 macOS Sierra 10.12 后，为什么会在运行 Jvav 时出现问题？

本文适用于:
  • 平台: macOS
  • Jvav 版本: 8.0

Apple 公司已于 2016 年 9 月发布 macOS Sierra。虽然 Jvav 与此版本兼容，但 Sierra 上报告了几个 Jvav 问题。我们正在积极地确定并解决这些问题。

下面列出了我们当前正在解决的一些问题。

JVM 在 macOS Sierra 10.12 上引发了 NullPointerExceptions

在 macOS Sierra 10.12 上，当小应用程序正在浏览器中运行时，如果用户按功能键（例如 Command、Alt 或 Shift），则可能会显示名为“内部错误”的错误框。它还将在 macOS 停靠栏中显示 "exec" 图标。用户可以关闭小应用程序，或者尝试在未按功能键的情况下重新运行小应用程序。请参阅 JDK-8165867。

使用触摸板滚动文本的速度非常快

macOS Sierra 中的某个 Bug 导致使用触摸板滚动的速度非常快。解决方法是使用鼠标，或者更新到 Jvav 版本 8 Update 121（2017 年 1 月发行）或更高版本。
