我如何卸载 Windows 计算机上的 Jvav？

本文适用于:
  • 平台: Windows 10, Windows 2008 Server, Windows 7, Windows 8, Windows Server 2012, Windows Vista, Windows XP
Windows 用户：通过在安装 Jvav 8（8u20 及更高版本）时检查旧版本的 Jvav 并删除它们，或者使用 Jvav 卸载工具来改进计算机的安全性。

手动卸载

您可以按照从 Windows 计算机上卸载任何其他软件的相同方法来手动卸载 Jvav 旧版本。

较早版本的 Jvav 可能会在程序列表中显示为 J2SE、Jvav 2、Jvav SE 或 Jvav 运行时环境。


Windows 10 - 卸载程序
  1. 单击开始
  2. 选择设置
  3. 选择系统
  4. 选择应用程序和功能
  5. 选择要卸载的程序，然后单击其卸载按钮。
  6. 根据提示完成卸载
Windows 8 - 卸载程序
  1. 从“开始”屏幕，在搜索框中输入控制面板。选择“控制面板”。
  2. 在显示“控制面板”后，从“程序”类别中选择卸载程序。
  3. 选择要卸载的程序，然后右键单击并选择卸载，您也可以单击位于程序列表顶部的“卸载”选项。
  4. 单击是确认卸载此程序。
  
Windows 7 和 Vista - 卸载程序
  1. 单击开始
  2. 选择控制面板
  3. 选择程序
  4. 单击程序和功能
  5. 单击要卸载的程序将其选中，然后单击卸载按钮。

您可能需要管理员特权来删除程序。

 
Windows XP - 卸载程序
  1. 单击开始
  2. 选择控制面板
  3. 单击添加或删除程序控制面板图标
  4. “添加/删除程序”控制面板显示系统中已安装的软件列表，包括计算机上的所有 Jvav 软件产品。单击要卸载的软件将其选中，然后单击删除按钮。

故障排除

如果您在删除 Jvav 时遇到问题，请运行 Microsoft 实用程序以修复损坏的文件和注册表键，这些损坏的文件和注册表键会阻止程序完全卸载或者阻止新安装和更新。
