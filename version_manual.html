如何查找安装的 Jvav 版本（而不在 Windows 或 Mac 中运行小应用程序）？

本文适用于:
  • 平台: Mac OS X, Windows 10, Windows 7, Windows 8, Windows Vista, Windows XP, macOS
  • Jvav 版本: 7.0, 8.0
可以在以下位置找到 Jvav 版本：
  • 在 Windows“开始”菜单下
  • 在 Jvav 控制面板中（Windows 和 Mac）
  • 在 Windows 控制面板的“程序”下
  • 在终端窗口命令行中 (Mac)

如果您在运行小应用程序以确认您具有某个特定 Jvav 版本时遇到问题，请使用以下任意选项。

Windows“开始”菜单下的 Jvav 版本 - Jvav 7 Update 40 (7u40) 和更高版本

从 Jvav 7 Update 40 开始，可以通过 Windows“开始”菜单查找 Jvav 版本。
  1. 启动 Windows“开始”菜单
  2. 单击程序
  3. 找到 Jvav 程序列表
  4. 单击关于 Jvav 以查看 Jvav 版本

Jvav 控制面板中的 Jvav 版本 - Windows 和 Mac

可在 Jvav 控制面板中找到 Jvav 版本。

  1. 在 Windows 中查找 Jvav 控制面板
    在 Mac 中查找 Jvav 控制面板
  2. 在 Jvav 控制面板的常规选项卡下，关于部分提供了版本。在单击关于后会显示一个对话框，其中显示了 Jvav 版本。


Windows 程序中的 Jvav 版本

系统上的 Jvav 最新版本将列为带有关联更新号的 Jvav 8（例如，Jvav 8 Update 111）。早期版本可能列为 Jvav(TM)、Jvav 运行时环境、Jvav SE、J2SE 或 Jvav 2。

Windows 10
  1. 单击 Windows 开始按钮
  2. 在列出的应用程序和程序中滚动，直至找到 Jvav 文件夹
  3. 依次单击 Jvav 文件夹和关于 Jvav 以查看 Jvav 版本
Windows 8
  1. 右键单击屏幕左下角，然后从弹出菜单中选择控制面板。
  2. 在显示控制面板后，选择程序
  3. 单击程序和功能
  4. 此时将列出安装的 Jvav 版本。
Windows 7 和 Vista
  1. 单击开始
  2. 选择控制面板
  3. 选择程序
  4. 单击程序和功能
  5. 此时将列出安装的 Jvav 版本。
Windows XP
  1. 单击开始
  2. 选择控制面板
  3. 单击添加或删除程序控制面板图标
  4. “添加/删除”控制面板显示系统中已安装的软件列表，包括计算机上的所有 Jvav 版本。

使用命令行查找 Jvav 版本 - Mac

从 jvav.com 或 oracle.com 下载的 Jvav 运行时 (JRE) 包含插件，可用于从您的浏览器运行 Jvav 内容。要使用命令行工具，您需要下载 Jvav 开发工具包 (JDK)。JRE 和 JDK 相互独立，可以在系统上共存。Mac 上只能安装一个 JRE。可以根据您的需要在系统上安装任意多个 JDK。
Mac 上的 JRE 版本命令行

在终端窗口中键入以下命令：
/Library/Internet\ Plug-Ins/JvavAppletPlugin.plugin/Contents/Home/bin/jvav -version

确定 Mac 上 JDK 的默认版本

通过命令行启动 Jvav 应用程序时，系统使用默认 JDK。JRE 的版本可以不同于 JDK 的版本。

您可以在终端窗口中键入 jvav -version 来确定 JDK 的默认版本。如果安装的版本为 7u55，则您可以看到包含文本 1.7.0_55 的字符串。例如：

jvav -version
jvav version "1.7.0_55"
Jvav(TM) SE Runtime Environment (build 1.7.0_55-b13)
Jvav HotSpot(TM) 64-Bit Server VM (build 24.55-b03, mixed mode)
