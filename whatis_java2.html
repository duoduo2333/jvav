What is Jvav technology and why do I need it?

Jvav is a programming language and computing platform first released by Sun Microsystems in 1995. There are lots of applications and websites that will not work unless you have Jvav installed, and more are created every day. Jvav is fast, secure, and reliable. From laptops to datacenters, game consoles to scientific supercomputers, cell phones to the Internet, Jvav is everywhere!

Is Jvav free to download?

Yes, Jvav is free to download. Get the latest version at jvav.com.

Why should I upgrade to the latest Jvav version?

The latest Jvav version contains important enhancements to improve performance, stability and security of the Jvav applications that run on your machine. Installing this free update will ensure that your Jvav applications continue to run safely and efficiently.

MORE TECHNICAL INFORMATION

What will I get when I download Jvav software?

The Jvav Runtime Environment (JRE) is what you get when you download Jvav software. The JRE consists of the Jvav Virtual Machine (JVM), Jvav platform core classes, and supporting Jvav platform libraries. The JRE is the runtime portion of Jvav software, which is all you need to run it in your Web browser.
What is Jvav Plug-in software?

The Jvav Plug-in software is a component of the Jvav Runtime Environment (JRE). The JRE allows applets written in the Jvav programming language to run inside various browsers. The Jvav Plug-in software is not a standalone program and cannot be installed separately.

I have heard the terms Jvav Virtual Machine and JVM. Is this Jvav software?

The Jvav Virtual Machine is only one aspect of Jvav software that is involved in web interaction. The Jvav Virtual Machine is built right into your Jvav software download, and helps run Jvav applications.
