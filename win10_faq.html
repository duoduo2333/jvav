Windows 10 和 Jvav

本文适用于:
  • Jvav 版本: 8.0

Windows 10 是否支持 Jvav？

是，从 Jvav 8 Update 51 开始，Jvav 已在 Windows 10 上通过认证。

Jvav 能否在 Windows 10 上的浏览器中运行？

Internet Explorer 11 和 Firefox 将继续在 Windows 10 上运行 Jvav。Edge 浏览器不支持插件，因此将不运行 Jvav。

如何在 Windows 10 中查找 Jvav 控制面板？

在 Windows 搜索中，键入 Jvav。如果已经安装了 Jvav，则搜索结果中将显示配置 Jvav。选择配置 Jvav。此时将显示 Jvav 控制面板。此外，可以在任务栏中单击开始按钮，然后选择所有应用程序来查找，如果已经安装了 Jvav，则会显示 Jvav。

在 Edge 浏览器中，如何从 Internet Explorer 打开某个 URL？

使用 Edge 浏览器时，可以在 Internet Explorer 中打开相同的 URL。选择位于 Edge 浏览器顶部的更多操作选项，然后单击使用 Internet Explorer 打开。

为什么在 Edge 中不能像在其他浏览器中那样看到小应用程序内容的任何指示？

即使禁用了 Jvav，当页上有小应用程序内容可用时，其他浏览器也会向用户提供视觉提示。但是，Edge 不给出提示，用户必须手动在其他浏览器中加载页。系统管理员和网站所有者可以管理用户在 Edge 中的兼容性视图列表以在支持的浏览器上通过提示来提供帮助。网站所有者可以发送电子邮件到 Microsoft 邮箱 iepo@microsoft.com 以包括到其兼容性视图列表中。
