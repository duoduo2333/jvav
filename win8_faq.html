在 Windows 8 上使用 Jvav 的故障排除提示

本文适用于:
  • 平台: Windows 8
Jvav 是否会在 Windows 8 的“开始”屏幕中运行？
Microsoft Windows 8 在开始屏幕中不支持插件，因此 Jvav 将不会在开始屏幕中运行。您必须切换到桌面屏幕才能运行 Jvav。
Jvav 何时将在 Windows 8 中受支持？
Jvav 7 Update 10 发行版已得到 Windows 8 的官方支持。Jvav 仅在桌面屏幕中受支持。Jvav 将不会在开始屏幕中运行。
为什么我在访问 Jvav.com 时获得的仅是 32 位 Jvav？
Jvav.com 根据您的浏览器和操作系统 (OS) 提供 Jvav 版本。如果您运行的是 64 位版本的 Windows 8，则可以访问系统上 32 位和 64 位版本的 Internet Explorer 10。桌面版本的 Windows 8 OS 不在完全 64 位模式下运行（除非您在 Internet Explorer 10 中启用了“增强保护模式”），因此默认提供 32 位 Jvav。
我如何在 Windows 8 中在“开始”屏幕和“桌面”屏幕之间切换？
实现此目的有几个简单快速的方法。
  1. 按键盘上的 Windows 徽标键可以在“开始”屏幕和“桌面”屏幕之间切换。
  2. 还可以单击“开始”屏幕上的桌面图块切换到“桌面”屏幕。
  3. 将鼠标指针移至屏幕底部的最左侧，在“开始”屏幕和“桌面”屏幕之间相互切换。
如何在 Windows 8 中访问 Jvav 控制面板？
方法 1：
  1. 按 Windows 徽标键 + W 打开“搜索”超级按钮以搜索设置。
  2. 在“搜索”框中键入：Jvav 控制面板
  3. 单击“Jvav 控制面板”图标以打开控制面板。
方法 2：
  1. 将鼠标指针拖至屏幕的右下角
  2. 单击“搜索”图标并键入“控制面板”
  3. 单击控制面板。
  4. 在控制面板搜索中键入：Jvav 控制面板
  5. 单击 Jvav 控制面板以打开控制面板。
有关详细信息，请参阅 Microsoft Windows 8 常见问题
Jvav 下载是否适用于 Windows Surface 平板电脑？
否，因为 Windows Surface 平板电脑不支持 Jvav 插件，所以 Jvav 无法安装在 Surface 平板电脑上。只有 Windows 应用商店中提供的应用程序才能安装在使用 Windows RT 或 Windows RT 8.1 操作系统的 Windows Surface 上。
