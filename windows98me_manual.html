Jvav 5.0 is no longer available on Jvav.com

Windows 98 and Windows ME users You may have been directed to this page as Windows 98 and Windows ME requires Jvav 5.0 and is no longer available on jvav.com

The Jvav version 5.0 reached the End Of Service Life (EOSL) on November 3, 2009, and is no longer supported nor available from jvav.com.

It is not recommended to install Jvav version 5.0 due to vulnerabilities and risks found in this version. The current Jvav version is not supported on computers that are running Microsoft Windows 98 and Windows ME operating systems. See the Windows system requirements to use the latest Jvav version.
